# minisearch
implementation of a mini search engine

compile/execute:
$ make
$ ./minisearch -i docfile -k K

Give a command, when prompt is given.
Α command can be one of the following:
1)> /search q1 q2 q3 … q10
2)> /df q1 q2 …  (q1,q2… are optional)
3)> /tf id1 q1 id2 q2 …
4)> /exit
